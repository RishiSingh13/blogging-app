# Anonymous Blogging Platform

## Description

An anonymous blogging platform built using the MERN stack where users can freely share their thoughts and ideas without revealing their identity. The application provides secure authentication and authorization using JWT (JSON Web Tokens) and cookies.

Users can create an account, log in securely, create and edit posts, like posts from other users, and browse content shared by the community. All posts are stored in MongoDB and displayed anonymously, encouraging open and honest expression.

## Features

* User Registration and Login
* JWT-based Authentication and Authorization
* Secure Cookie Management
* Create Posts
* Edit Existing Posts
* Like and Unlike Posts
* View Posts from Other Users
* Anonymous Posting System
* MongoDB Database Integration
* Responsive User Interface

### Database

### Authentication

* JWT (JSON Web Tokens)
* Cookies
* bcrypt Password Hashing

## How It Works

1. Users create an account or log in.
2. Authentication is handled using JWT tokens stored in cookies.
3. Authenticated users can create and manage posts.
4. Posts are stored in MongoDB.
5. Users can view and like posts shared by others.
6. The platform keeps posts anonymous, allowing users to express themselves freely.
